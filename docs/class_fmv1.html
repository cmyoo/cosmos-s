Detailed Description

Derived class representing a refinement grid (Layer > 0). @inherits Fmv0.

Specializes Fmv0 for a refinement level in the FMR hierarchy. Contains pointers to the parent (llay) and child (ulay) layers. Implements boundary conditions that involve interpolation from the parent layer and restriction to the parent layer. Manages the time stepping synchronization between layers.
