Checks parameter consistency when continuing a simulation.

Reads header information from the continuation file and compares key parameters (like fluid/scalar flags, excision settings, grid setup) against the current configuration read from parameter files. Exits if inconsistencies are found.

Parameters
fcontinueInput file stream connected to the continuation data file.
fldCurrent fluid evolution flag (to be checked).
sclCurrent scalar evolution flag (to be checked).
excExcision flag read from the continuation file (output).
exgCurrent excision grid setting (to be checked against file).
tabCurrent buffer grid size (to be checked).
nzmaxCurrent base layer z-grid size (to be checked).
laymaxCurrent maximum allowed FMR layers (to be checked against file).
lnNumber of FMR layers found in the continuation file (output).
lbsArray of FMR boundary indices (to be checked).

int main ( int argc,

Main entry point for the COSMOS-S simulation.

This function orchestrates the entire simulation process. It reads initial parameters, sets up the simulation grid(s) potentially with Fixed Mesh Refinement (FMR), handles continuation from previous runs, executes the main time evolution loop, manages output, and performs final cleanup.

Parameters
argcNumber of command-line arguments.
argvArray of command-line argument strings (currently unused).
Returns
0 on successful completion, non-zero on error.
